## 🖼 Slide 4: - 4 -

  - 선택관광 노출 기본 정책
    - 적용대상
      - 에어텔 상품, 세미패키지 상품 중 테마구분이 ‘퍼즐팩’으로 설정된 상품에 한해 적용된다.
      - 허브 판매상품 상세에 등록된 선택관광 리스트 내 ‘퍼즐팩’으로 설정된 선택관광에 한해 추가 가능하다.
        - 테마가 ‘퍼즐팩’인 상품은 허브의 판매 상품상세에서 일정에 등록된 선택관광과 선택관광 리스트에 등록된 선택관광이 서로 연동된다.
      - 허브 판매상품 상세 내 선택관광 리스트에서 스페셜로 설정된 선택관광은 제외된다.
    - 선택관광 노출, 추가 정책
      - 선택관광 추가 기능은 예약하기 페이지와 예약내역상세 페이지는 기존 정책 유지하며, 세 페이지 간 선택 내역은 동일하게 연동된다.
      - 예약내역상세 페이지(예약완료 후)에서는 추가된 선택관광은 취소가 불가하고 추가만 가능하다.
      - 예약이후 선택관광 취소, 변경 불가, 유선으로 가능
      - 일정표에서 선택된 선택관광은 예약하기에도 동일하게 보이며, 삭제 / 수정/ 추가 가능하다.
      - 예약하기 페이지에서 Back 버튼을 실행하면 모든 선택 내역은 초기화된다.(기존 정책 동일),
      - ‘퍼즐팩’ 테마 상품의 경우 일정표에 선택관광 노출 정책과 예약하기, 예약내역상세 페이지도 동일한 정책을 적용한다.
      - 선택관광추가는 ‘선택관광추가하기’버튼을 통해서 추가 가능하며, 기존 선택관광 리스트 노출은 유지한다.
      - 추가는 선택관광 기준으로 인원을 설정한다. (기존 정책 동일)
      - 성인, 아동만 선택 가능하다. (유아 제외)
      - 일차별 동일 선택관광 중복선택 불가하다. (1차별 1개의 각각 다른 선택관광만 선택 가능)
      - 선택관광의 환율정보는 KB하나은행에서 정기 배치로 가져오는 정보이다.- 고시 된 환율 중 ‘최초’고시 기준으로 계산 (허브 > 업무공통 > 기준정보 > 기초정보관리 > 기준환율조회 > 최초)
      - 선택관광 금액에는 쿠폰 및 할인 이벤트가 적용되지 않는다.
- 퍼즐팩 Phase1 - 일정표 선택관광 추가  정책

## 🖼 Slide 7: - 7 -

      - 선택관광 추가하기 (레이어 반팝업)
        - 1개의 선택관광에 대해 인원 설정이 가능하다.
        - 일정표 내 일정상세에서 노출되는 선택관광의 하단 버튼을 통해 호출된다.
        - 추가 가능한 선택관광 리스트가 노출되는 팝업에서 선택관광 선택 시 호출된다.
        - 팝업 내 노출 되는 정보는 선택관광 카드매니저 내용이며, 설정된 카드매니저에 따른 템플릿으로 노출된다.
          - 정보 노출 되는 순서는 카드매니저 기준으로 노출 된다.
            - 선택관광명, 선택관광 이미지 (2개 이상인 경우 페이지 네이션 추가), 선택관광 설명, 인원설정 영역
            - 인원설정은 성인, 아동만 가능하며 금액은 원화, 현지화로 노출된다.
        - 인원선택 Default 값은 0 이며 0인 경우 ‘–’값은 비활성화 처리된다.
        - 최대 선택 가능 인원은 상품에서 선택한 인원수이다.
        - 선택 인원수보다 많은 인원수를 선택하려고 할 경우 알럿이 호출된다.
        - 상품의 설정된 인원수 보다 선택관광의 인원수가 상이한 경우
          - 상품 인원 수 < 선택관광 인원 수 : [+]버튼 비활성화, 알럿 노출
            - 알럿 문구 : 상품 인원수보다 많은 인원수는 선택 할 수 없습니다.
          - 상품 인원 수 > 선택관광 인원 수 : 여행자 순번으로 적용
            - 예시) 상품 인원수 = 성인4명, 선택관광 인원수 = 성인2명인 경우 여행자1(성인), 여행자2(성인)에 적용된다.
        - 하단 선택 완료버튼은
          - 인원 설정 내역이 없을 경우 : 버튼 비활성화 (닫기) , 실행 시 팝업 닫힘
          - 인원 설정 내역이 있을 경우 : 버튼 활성화 (담기 (+선택관광 인원 설정 합산 금액)
