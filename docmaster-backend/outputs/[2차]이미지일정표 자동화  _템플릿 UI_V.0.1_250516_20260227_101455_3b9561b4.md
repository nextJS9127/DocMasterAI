## 🖼 Slide 3

- 개요

- 1. 작업 배경 및 목적

- - 배경 : 상품 정보를 제공하는 콘텐츠에 대한 정제된 정보 필요성 제고 - 목적 : 기존 상품담당자가 제공하는 템플릿안에서 AI 도구를 활용하여 일정표를 간편하게 제작

- 2. 작업 방안

- 허브 내 이미지 일정표 템플릿 메뉴 신설- 상품담당자가 이미지, 동영상 등의 멀티미디어 정보를 활용한 콘텐츠 자유롭게 생성- 타 메뉴와의 연계성 강화

- 허브 내 이미지 일정표 별도 운영 메뉴 신설- 이미지 일정표 작성 에디터 및 관리 기능 강화

- AI를 활용하여 영상 제작 및 일정표 생성

- 3. 기대효과

- - 상품담당자가 이미지 일정표를 직접 제작하여 기존 운영 프로세스 및 리소스 간소화

- - AI 도구를 적극 활용함에 따라 생산성 제고

- 4. 작업 범위 및 일정

- 하나허브 내 관리자 메뉴 신설 후 닷컴 PC / MO 제공_Phase1 완료 (5.9 현업 메뉴 권한 오픈 완료)

- 템플릿  확대 (N박 상품, 밍글링투어, 패키지 템플릿)_Phase2
